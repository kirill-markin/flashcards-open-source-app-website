## تطبيق الويب المستضاف

أسرع طريقة للبدء هي تطبيق الويب المستضاف:

1. افتح [app.flashcards-open-source-app.com](https://app.flashcards-open-source-app.com)
2. سجّل الدخول ببريدك باستخدام OTP بلا كلمة مرور
3. أنشئ بطاقات وراجع العناصر المستحقة واستخدم دردشة الذكاء الاصطناعي مع بيانات مساحة العمل والملفات المرفقة

لا يتطلب المسار المستضاف أي تثبيت أو إعداد خادم.

## إعداد الوكيل

إذا كنت تريد أن يتصل Claude Code أو Codex أو OpenClaw مباشرة، فابدأ من:

```text
GET https://api.flashcards-open-source-app.com/v1/
```

تشرح استجابة الاكتشاف هذه للوكيل كيفية تسجيل الدخول عبر OTP بالبريد، وإنشاء مفتاح API طويل العمر، وتحميل الحساب، وتهيئة مساحة العمل، ثم متابعة سطح SQL المنشور.

الحمولة نفسها متاحة أيضًا عبر `GET /v1/agent`، لكن `/v1/` هو نقطة الدخول العامة الأساسية.

## الاستضافة الذاتية

إذا كنت تفضّل تشغيل نسختك الخاصة، فراجع [دليل الاستضافة الذاتية](/docs/self-hosting/).

## ما الذي تحصل عليه اليوم

- تطبيق ويب مستضاف للبطاقات والمراجعة ودردشة الذكاء الاصطناعي
- عميل iOS في المستودع الرئيسي مع SQLite محلي ومزامنة تعمل دون اتصال أولًا
- خدمات خلفية ومصادقة مشتركة على نطاقي `api` و`auth` المنفصلين
- إعداد خارجي للوكلاء عبر الاكتشاف وOTP ومصادقة ApiKey
- مسار نشر مفتوح المصدر على AWS مع Postgres كمصدر الحقيقة

## اتجاه المستودع

المشروع مصمم وفق مبدأ العمل دون اتصال أولًا.

اليوم يتضمن المستودع تطبيق الويب وتطبيق iOS وخدمة المصادقة وواجهة backend API وتدفق الوكيل الخارجي وتطبيق Android المنشور على Google Play.
