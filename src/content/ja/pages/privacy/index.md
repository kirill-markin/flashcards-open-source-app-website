## 収集する情報

ホスト型クラウドベータを利用する場合、認証に使用するメールアドレスと、サービス提供に必要な学習データを保存します。学習データには、カード、復習履歴、ワークスペースのメタデータが含まれます。セルフホスト環境では、ご自身でそのように構成したコピーをデプロイしない限り、当社にデータが送信されることはありません。

## AI 機能

ホスト型アプリで AI チャットを利用する場合、入力したプロンプト、リクエスト処理に必要なカードに基づくコンテキスト、アップロードしたファイルや画像、ならびに音声入力データや文字起こしリクエストが、サーバーに設定された第三者の AI プロバイダーへ送信されることがあります。実際に使用されるプロバイダーは、リクエスト時点のホストサーバーの設定によって異なります。

## 外部 AI クライアントとの接続

リモート MCP サーバー（`mcp.flashcards-open-source-app.com/mcp`）または Agent API を通じて Flashcards を外部の AI クライアントに接続すると、あなたがリクエストしたカードデータは、そのクライアントおよびそれを運用する AI またはモデルのプロバイダーへ送信されます。この処理は当社ではなく、そのプロバイダーの規約に従います。

## データの利用目的

あなたのデータは、認証、クラウド同期、および任意で利用できる AI 機能の提供のために使用されます。当社があなたのデータを販売したり、広告目的で利用したりすることはありません。認証とワークスペースへのアクセスは、公開リポジトリに記載されているバックエンドサービスによって処理されます。

## データの保存

ホスト型クラウドのデータは AWS 上のインフラに保存され、主要なアプリケーションデータには Postgres を使用しています。データは転送中に暗号化されます。ホスト型の学習データは、アカウントが有効な間は保持し、アカウントを削除した場合または削除をリクエストした場合に削除します。運用ログは、ホスト型ベータの運用やデバッグに必要な期間に限り保持し、通常は最長 90 日です。

## クッキー

ログイン処理のために、`session`、`refresh`、`logged_in` などの認証クッキーを使用します。サイトの動作に、トラッキングクッキーや第三者のアクセス解析は必要ありません。

## データの削除

セルフホスト環境では、データベースを直接管理できます。ホスト型アプリでは、iOS アプリまたはウェブアプリからアカウントを削除できます。ホスト型データの削除は、Agent API または接続済みの MCP クライアントからリクエストすることもできます。ホスト型データの削除について追加のサポートが必要な場合は、お問い合わせください。
