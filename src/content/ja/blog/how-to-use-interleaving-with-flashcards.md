合成関数の微分法（chain rule）を問うフラッシュカードを10枚続けて解けば、その解法は簡単に思えてきます。ところが、合成関数の微分法・積の微分法・商の微分法のどれを使うべきか自分で判断する混合問題になると、さっきまでの自信が消える。その差をあぶり出すのが、インターリービング（交互練習、interleaving）です。

**フラッシュカードでインターリービングを使う方法**は、まず新しいカテゴリーを見分けられる程度まで個別に学び、その後、関連する2つか3つのカテゴリーを1回の想起練習に混ぜることです。カテゴリーを明かすラベルは外し、カードをめくる前に答えます。各カードを次にいつ復習するかはFSRSに任せてください。見た目に変化をつけるためだけに、無関係なデッキを混ぜてはいけません。

インターリービングが変えるのは、次にどの種類の問題やカードが出るかです。間隔反復が変えるのは、同じカードがいつ戻るかです。解決する問題が異なるため、この2つはうまく組み合わせられます。

## インターリービングと間隔反復の違い: 並び順とスケジュール

[デューク大学による実用的な定義](https://arc.duke.edu/interleaving/)では、インターリービングを、学習セッション中に複数のトピックを意図的に混ぜることと説明しています。一方、ブロック練習（blocked practice）は、同じ種類の内容をまとめて練習する方法です。

`A`、`B`、`C`が関連する問題タイプなら、2つの並び方は次のようになります。

- ブロック練習: `A A A B B B C C C`
- インターリービング: `A B C B A C A C B`

インターリービングでは、次々と選択を迫られます。手順を適用する前に、まず問題の種類を見極めなければなりません。ブロック練習では、直前のカードが次に使う解法の種類をすでに教えています。

間隔反復は別の軸で働きます。記憶が安定するにつれて、あるカードが翌日、4日後、その次は2週間後に戻ってくることがあります。FSRSは、復習履歴をもとにそのタイミングを管理します。

| 方法 | 答える問い | 変わるもの |
| --- | --- | --- |
| アクティブリコール | 今、答えを取り出せるか | 裏面を見る前に答える |
| インターリービング | 次にどの種類の知識が必要か | 関連するカードや問題の種類を混ぜる |
| FSRSを使った間隔反復 | このカードを次にいつ戻すか | 復習間隔が変わる |

復習期限を迎えたカードのキューに複数のトピックが入っていても、それだけで意図的なインターリービングになるわけではありません。単に同じ日に復習期限が来ただけかもしれません。有益なインターリービングにするには、カテゴリーを同じセットに入れる理由が必要です。互いに混同しやすい、競合するルールを使う、試験でどれかを選ぶ必要がある、といった理由です。

調査対象となった学生の多くは、この用語自体を知りませんでした。[2大学の学部生1,093人を対象にした2025年の調査](https://onlinelibrary.wiley.com/doi/full/10.1002/acp.70071)では、インターリービングという用語の認知度が低く、意識的な利用も限られていました。ただし、名前を知らないまま、古い内容と新しい内容をすでに混ぜていた学生もいました。

## ブロック練習のほうが手応えを感じやすい理由

ブロック練習では、カードごとに必要な判断が1つ減ります。直前の6問で積の微分法を使っていたなら、7問目もおそらく同じです。練習中の成績はすぐに上がり、すらすら進んでいるように感じます。けれど、その感覚が示しているのは、知識を単独で使えることではなく、並び順に適応したことかもしれません。

[259人が6人の画家の作品を分類する課題に取り組んだ2025年の実験](https://www.sciencedirect.com/science/article/pii/S1041608025001803)では、学習中の成績と「学べている」という自己評価は、ブロック練習のほうが高くなりました。一方、学習直後と時間を置いたあとの成績は、ランダム型と適応型のインターリービングのほうがブロック練習を上回りました。この課題では、適応型の並び順が通常のランダム型インターリービングを上回ることはありませんでした。

この結果だけで、どんな科目でも、すべてのフラッシュカードデッキをシャッフルすれば効果が上がるとは言えません。ただし、落とし穴はよく見えます。特に似たカテゴリーを見分ける課題では、練習中の成績と後の学習成果が別の方向を向くことがあります。

教室での研究を見ると、想起練習とのつながりがさらに分かりやすくなります。[高校の理科を学ぶ生徒155人を対象にした2022年の研究](https://journals.sagepub.com/doi/10.1177/09567976211057507)では、4週間にわたり、毎週のクイズ問題を概念ごとにまとめるか、複数の概念を混ぜて出題しました。1か月後のテストでは、インターリービング形式のクイズで練習した概念のほうが、ブロック形式で練習した概念よりよく記憶されていました。これは教室で行われたクイズであり、FSRSのフラッシュカードキューではありません。そのため、この結果は複数の概念を混ぜた想起練習を支持しますが、特定のアプリ内の並び順まで証明するものではありません。

選択と想起に由来する難しさなら、学習に役立つことがあります。分かりにくい文面、長すぎる答え、無関係な文脈への切り替えは、単なる余計な負荷です。

## 勉強でトピックを混ぜる方法: 競合する判断を選ぶ

インターリービングに最も向いているのは、互いに関連し、違いを見抜く必要がある内容です。

相性のよい例は次のとおりです。

- 積、商、合成関数の微分法を使う問題
- 需要曲線上の移動と、需要曲線そのもののシフト
- 短い文脈におけるスペイン語の点過去と線過去
- 結果が示された設問における有糸分裂と減数分裂
- 入力は似ているが保証内容が異なる、2つのプログラミング手法

向いていないのは、スペイン語の語彙カード、有機化学の反応機構、税務上の期限、ネットワークコマンドを1つの山に放り込むような組み合わせです。変化は生まれても、有益な比較にはなりません。

[インターリービングに関する59件の研究を分析した2019年のメタ分析](https://doi.org/10.1037/bul0000209)は、慎重に考えるべき理由を示しています。全体では中程度の効果が見られましたが、結果は教材に大きく左右されました。視覚的なカテゴリー学習では効果が強く、数学課題では全体として効果が小さく、説明文では結果が曖昧でした。単語を使った研究では、全体としてブロック練習のほうが優位でした。カテゴリーどうしの類似性も、効果を左右する重要な要因の1つでした。

したがって、実用上の判断基準は「このトピックをシャッフルできるか」ではありません。「このトピックを切り替えることで、適切なルール、カテゴリー、説明を自分で見極める必要が生まれるか」です。

| 混ぜる | 今は分けておく |
| --- | --- |
| 何度も取り違える似たカテゴリー | まだ理解できていない内容 |
| 試験で一緒に出る問題タイプ | 長い読み物や初回の解説 |
| 適用条件が異なる、競合するルール | 共通する判断がない、無関係な科目 |
| 同じ区別を問う古い例と新しい例 | 文面だけでカテゴリーが分かるカード |

## ブロック練習とインターリービング: いつ切り替えるか

最初の例からすべてインターリービングにすると、必要以上に厳しい学習になります。商の微分法を一度も見たことがないのに、1行学んだだけで別の内容へ移っても、見分ける力は身につきません。解説が途中で切れるだけです。

ブロック練習からインターリービングへ進めば、両方の有用な部分を活かせます。

### 1. 1つのカテゴリーを、名前を言える程度まで学ぶ

解説を読み、同じ種類の例を数問解きます。目標は速さではなく、基本をつかむことです。そのルールが何をするものか説明でき、単純な例を見分けられる状態を目指します。

2つ目のカテゴリーでも、この短い導入を繰り返します。最初の2つの基本が分かってから、3つ目を加えます。

この最初の段階が重要なのは、事前知識によって結果が変わり得るからです。[ドイツの小学3年生108人を対象に事前登録された2025年の教室研究](https://www.sciencedirect.com/science/article/pii/S0959475225000702)では、似た綴りの規則をインターリービングで練習すると、新しい単語への転移を含む持続的な効果が見られました。ただし、その効果が主に見られたのは、綴りに関する事前知識が平均から高い子どもでした。子どもたちはインターリービング中に間違いも増え、自分の学習成果をより厳しく評価しました。この研究は、その条件でのインターリービングを支持すると同時に、どんな学習でも最初から使える万能な方法ではないことも示しています。

### 2. 混同しやすいカテゴリーで小さなセットを作る

関連するカテゴリーを2つか3つ選びます。まずは各カテゴリーから3枚か4枚ずつといった、控えめな量から始めます。この枚数は実用上のひな型であり、研究で確立された閾値ではありません。

一時的な練習カードや、カード候補の問題文を使っても構いません。まだ長期的な復習キューに入れる必要はありません。

整理に役立つなら、カテゴリー名をメタデータ、ノート、準備用リストに残しておきます。ただし、カテゴリーそのものを見抜く練習なら、カードの表面には表示しないでください。

`合成関数の微分法: 次を微分せよ...` では、どの判断をすべきかが見えています。

`次を微分せよ... 最初に使う微分法は？` なら、自分で選ばなければなりません。

### 3. 短い混合練習を1セット行う

一時カードを自分でシャッフルするか、カテゴリーが切り替わるように並べます。`A B C A B C`という規則的な繰り返しが答えの手がかりになり始めるなら、その並びは避けます。

どのカードも、裏面を見る前に表面の問いへ答えます。手順を問うカードなら、まずルール名を答え、そのあとで計算や作業を進めます。概念カードなら、カテゴリー名に加え、最も近い別のカテゴリーと見分ける手がかりも答えます。

このセットは、追加のスケジュール復習ではなく、弁別練習として扱います。保存済みのカードがまだ復習期限を迎えていないなら、混合セットに入れるためだけに評価を送信しないでください。復習履歴が変わってしまいます。練習には問題文だけをコピーして使い、評価を記録するのは通常の復習キューだけにします。

### 4. 何度も混同する内容を対比カードにする

通常のカードは、1つの答えを確認します。対比カードは、2つの答えの境界を確認します。

たとえば次のようなカードです。

- 表面: ほかの条件が一定のまま、ある商品の価格が下がった。これは需要曲線のシフトか、それとも需要曲線上の移動か。
- 裏面: 需要曲線上の移動。商品の価格自体が変化すると需要量が変わり、それ以外の決定要因が変化すると需要曲線がシフトする。

このカードが役立つのは、`需要曲線のシフト`の定義だけでなく、どちらを選ぶかという判断ルールを残せるからです。

裏面は、正直に自己採点できる長さに保ちます。[より良いフラッシュカードの作り方](/ja/blog/how-to-make-better-flashcards/)では、こうしたカードの書き方を詳しく扱っています。[アクティブリコールと間隔反復の違い](/ja/blog/active-recall-vs-spaced-repetition/)では、想起とその後のスケジュール管理が別の仕事である理由を説明しています。

### 5. 残す価値があるカードはFSRSに任せる

練習後、その区別が今後も重要な場合や、何度も間違えた場合にカードを保存します。インターリービングを理由に、復習期限を無視したり、すべてのトピックを毎回の復習セッションへ手動で押し込んだりしてはいけません。問いが明確なカードを長期復習に入れたら、正直に自己採点し、次の復習間隔はFSRSに決めてもらいます。

もう一度見分ける力を試したいときは、新しい例を使った短い練習用セットを別に作ります。練習例をすべてスケジューラに登録する必要はありません。

## インターリービングで使うフラッシュカードセットの実例

3つの微分法を学んでいるとします。まず短いブロック練習で、各微分法の単純な例を数問ずつ解きます。その後、カテゴリー名を外してカードを混ぜます。

カード1:

- 表面: `(x² + 1)⁴`を微分せよ。最初に使う微分法は？
- 裏面: 合成関数の微分法。導関数は`8x(x² + 1)³`。

カード2:

- 表面: `x²eˣ`を微分せよ。最初に使う微分法は？
- 裏面: 積の微分法。導関数は`2xeˣ + x²eˣ`。

カード3:

- 表面: `(x + 1) / (x - 1)`を微分せよ。最初に使う微分法は？
- 裏面: 商の微分法。導関数は`-2 / (x - 1)²`。

カード4:

- 表面: `sin(x³)`を微分せよ。最初に使う微分法は？
- 裏面: 合成関数の微分法。導関数は`3x² cos(x³)`。

この混合セットの役割は、単に変化をつけることではありません。どのカードでも、計算を始める前に式の構造を見極め、微分法を選ぶ必要があります。試験では、問題の上に`商の微分法`というラベルは付きません。その状況に近い練習です。

カード2で何度もつまずくなら、対比カードを1枚加えます。

- 表面: `x²eˣ`で、合成関数の微分法ではなく積の微分法から始めるのはなぜか。
- 裏面: `x`の関数を2つ掛け合わせているから。どちらか一方の関数が、もう一方の内側に入っているわけではない。

同じ例を少しだけ変えたカードを4枚も保存しないでください。見た目だけが違うカードでデッキを埋めるより、明確な対比カード1枚と、その後の混合練習のほうがたいてい役に立ちます。

## インターリービング用のフラッシュカードにもアクティブリコールを求める

上で紹介した絵画の研究が示すように、インターリービングはカテゴリー学習を助けることがあります。それでも、フラッシュカード自体は、身につけたい記憶に合う問い方でなければなりません。答えを自力で出さなくても、表面のヒントだけで裏面を見て正解だと分かるなら、そのカードを混ぜても想起カードには変わりません。

選択肢が4つあるカードなら、見慣れた言葉を見つけるだけで答えられるかもしれません。見出しにカテゴリー名が入っていれば、使うべき手順が分かってしまいます。ほぼ同じ例が5問続けば、並び順の記憶だけで答えられることがあります。

インターリービング用のフラッシュカードでは、少なくとも次のどれか1つを自分で取り出す必要があります。

- 答えそのもの
- 当てはまるカテゴリーやルール
- 近い別の選択肢と見分ける特徴
- 手順の最初の一歩

ここで、**インターリービングとアクティブリコール**が互いを補います。アクティブリコールが想起の試みにあたり、インターリービングは、次にどの種類の想起が来るか分かっている安心感を取り除きます。

実際の課題がカード1枚より大きいなら、模擬試験も必要です。複数ステップの作業、時間配分、別の問題への応用には混合問題を使い、そこで見つかった小さなルールや区別を保つためにフラッシュカードを使います。[フラッシュカードと模擬試験の違い](/ja/blog/flashcards-vs-practice-tests/)では、この使い分けをさらに詳しく説明しています。

## インターリービングを悪化させる、よくあるやり方

何もかも混ぜると、場当たり的な文脈切り替えになります。無関係なカテゴリーを並べても、共通する区別を自動的に学べるわけではありません。切り替えの手間が増えるだけで、有益な選択は練習できないことがあります。

始めるタイミングも重要です。知らないラベルの間で勘に頼って選ぶのではなく、カテゴリーを比較できるだけの理解が必要です。

カテゴリーを示すヒントが、気づかないうちに練習全体を台なしにすることもあります。デッキ名、見出し、繰り返される言い回し、固定された並び順のどれも、次に必要な答え方を明かしかねません。

練習速度だけでこの方法を評価しないでください。インターリービングは判断を1つ増やすため、遅く感じることがよくあります。その判断自体が練習の一部です。

適応型やAI生成の並び順なら必ず優れている、という主張には慎重になるべきです。2025年の絵画研究では、適応型の並び順に、ランダム型インターリービングを上回る学習上の利点は見られませんでした。この結果は1つの研究設計に限られます。それでも、個別最適化された順番が自動的に優れているという主張を退けるには十分です。

並び順を試すために、スケジューラを巻き込まないでください。インターリービングと間隔反復を組み合わせるときも、FSRSの復習履歴と期限は維持します。意図的な混合練習は、通常の復習システムとは別枠で加えてください。見栄えのよいパターンを保つために、十分に定着したカードを何度もリセットしたり、スケジュールし直したりしてはいけません。

## Flashcardsをこの学習フローに組み込む方法

[Flashcards Open Source App](/ja/features/)は、この学習フローで長期定着を担います。表面と裏面が明示されたカード、4段階の復習評価、期限を迎えたカードに対するFSRSのスケジュール管理を提供します。公開されている機能仕様では、デッキやワークスペースをまたいだ自動インターリービングを約束していません。そのため、通常の復習キューが意図的な対比ペアを組んでくれるとは考えないでください。

役割の境界を明確に保ちます。

1. ルール、区別、何度も間違える内容について、明確な表裏カードを作る。
2. 元資料が整理されていないときは、その資料をAIチャットに添付してカード候補の下書きを作り、すべての答えを自分で確認する。
3. カテゴリーを選ぶ練習が必要なときは、関連する問題文を自分で選んで混ぜ、できれば新しい例を使って通常の復習キューの外で練習する。
4. 残す価値がある誤答と、判断に時間がかかった区別だけをカードにする。
5. 期限を迎えたカードは通常どおり復習し、その後のタイミングはFSRSに更新させる。

この分担なら、よくある混乱を避けられます。インターリービングのセッションは弁別に集中し、FSRSのキューは長期的な定着に集中できます。

[FSRS設定ガイド](/ja/blog/fsrs-settings/)では、スケジューラの調整に凝りすぎず、目標保持率と復習負荷を決める方法を説明しています。カードを作り、期限を迎えた項目を復習する準備ができたら、[使い始めガイド](/ja/docs/getting-started/)から提供中のWebアプリへ進めます。

## 覚えておきたい実用的なルール

混ぜるのは、無関係な科目ではなく、競合する判断です。本当に新しいルールは、まずブロック練習から始めます。基本的なカテゴリーを見分けられるようになったら、インターリービングへ移ります。ラベルを外し、答えを思い出し、違いを説明し、なぜ間違えたかに気づける程度の小さなセットに保ちます。その後の復習時期はFSRSに任せます。

これが、**インターリービングと間隔反復**をすっきり組み合わせる方法です。インターリービングは、複数の選択肢が混ざった状況で適切な知識を選ぶ準備を整えます。一方、FSRSは、復習履歴と保持率の設定をもとに、残す価値がある各カードの次の復習時期を決めます。
