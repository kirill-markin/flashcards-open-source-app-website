上周我在用 Quizlet 学习，Learn 模式已经刷到第三轮，终于进入了那种卡片越翻越顺的状态。结果一个面板从底部滑了出来：我达到了本轮上限，想继续就得开通 Quizlet Plus。我已经好几年没为做测试花过钱了，可现在，连继续学自己做的卡组都要付费。

如果你正是因为这一刻而来到这里，那么确实有一款真正**免费的 Quizlet 替代方案**，而且它不是被砍得只剩骨架的试用版。[Flashcards](/) 是开源的，beta 期间托管方案免费使用，自托管则永久免费。所有功能在每个方案里都有，你真正用来学习的那些部分没有任何付费墙。下面我会讲讲 Quizlet 到底变了什么，以及为什么我认为这个对比是公平的，而不是一篇刻意抹黑的文章。

## Quizlet 上的“免费”过去意味着什么，又变了什么

很长一段时间里，免费版 Quizlet 其实相当好用。你可以做一套卡组，想刷多少遍 Learn 模式就刷多少遍，还能做练习测试，完全不用考虑订阅。正是这份慷慨，让那么多学生认识了它。

到了 2026 年，免费层缩水了。Learn 模式每次大约只能刷几轮，之后就会提示你升级；Test 模式每套卡组大约只能做一次。无限制的 Learn 和 Test 如今都被划进了 Quizlet Plus，每月大约 8 美元，促销时偶尔接近 3 美元。还有几样原本免费的功能也搬到了同一道付费墙后面：离线访问需要 Plus，给卡片加图片也需要 Plus。除此之外，免费层还带广告。

人们的不满不难理解。评论里有人称它“掠夺式”，说“我们现在真的得花钱才能学习了”。我还看到一条说它“只剩个空壳”。这些措辞很重，但背后是一个真实的转变：原本免费的那件事——学习本身——现在有一部分要收费了。

平心而论，Quizlet 仍然是个不错的起点。它快、够主流，而且几乎人人都已经会用。如果你只是想周五考试前快速做一套卡片，它完全够用。问题其实比“Quizlet 不行”要具体得多：是免费版不再让你想学多久就学多久。

## 一款真正免费的 Quizlet 替代方案应该意味着什么

“免费”这个词藏着不少陷阱。很多应用自称免费，却把你真正需要的部分锁了起来。做卡片免费，但间隔重复要花钱；或者学习免费，但每天只能学十张卡。试用版贴上“免费”标签，它依然是试用版。

一款真正的**免费 Quizlet 替代方案**，标准应该更严格：学习没有轮次上限，学到一半不会跳出升级提示，好算法是内置的而不是作为附加项另卖，离线学习无需订阅，卡片能带图片且不附带任何条件。这就是我在意的底线，也是这款应用从一开始就围绕它来做的。

## Flashcards 里真正免费的部分

[Flashcards](/) 提供两种使用方式，现在两种都免费。

Cloud Beta 层在 beta 期间免费。它是项目域名上的托管 web 应用，采用免密码邮箱登录。输入一个发到邮箱的一次性验证码，你就能开始学习。不用填卡片信息，也没有试用倒计时。

Self-Hosted 层完全免费。整套技术栈在 GitHub 上开源：web 应用、后端、认证，以及一个本地 Postgres 数据库。你用自己的域名、自己的数据、自己的部署来运行它。同一个产品，由你来托管。

对“免费”这个承诺最关键的一点写在 [定价页](/pricing/) 上：所有功能在每个方案里都能用，没有任何功能阉割。没有刷几轮就锁住的 Learn 模式，也没有为图片单独收费的条目。因为代码以公开许可证开源，作为一款开源 Quizlet 替代方案，免费的路径会永远存在。即便某个托管方案日后有变动，你也始终可以自己运行它——这比“暂时免费，相信我们”要踏实得多。

## FSRS 是对 Quizlet 收费那个 Learn 模式的升级，而它在这里免费

这正是 Learn 模式那道墙显得有点荒唐的地方。Learn 模式是个相当基础的排程器，把你的卡片打乱、推着你多看较薄弱的卡片，作为基础功能它没什么问题。而同一个思路的现代版本要好得多，Flashcards 免费就在用。

那个算法就是 FSRS。它根据你忘记每张卡的概率来决定何时再次展示它，于是简单的卡片间隔越拉越长，困难的卡片更快回来。在几百张卡的规模上，这就是复习队列“合理”还是“瞎忙活”之间的区别。

Quizlet 如今要收费才让你继续做的事，Flashcards 做得更好，而且分文不取。如果你想了解 FSRS 为什么胜过更旧排程器的技术原理，我在 [2026 年 FSRS vs SM-2](/blog/fsrs-vs-sm-2/) 里写过。

## 无需付费的离线学习

离线在 Quizlet 上曾经免费，如今需要 Plus。这挺让人难受，因为在火车上或飞机上学习，恰恰是你有时间却没信号的时候。

Flashcards 在 web 和 iOS 上都是离线优先的。卡片就存在你的设备上。你当下复习，答案立刻保存，等你重新联网再做同步。你不必为没有 Wi‑Fi 也能学习这件事付订阅费。

我会故意挑些恼人的环境来测试学习应用：信号差、电量低、没耐心。如果离线正是把你推离 Quizlet 的主要原因，[2026 年最佳离线抽认卡应用](/blog/best-offline-flashcards-app/) 会更深入地讲讲离线优先到底意味着什么。

## 把你的 Quizlet 卡组带过来，不必丢掉已有的成果

离开时一个真实的顾虑，是你已经投入的那些心血。那些卡组是你熬了许多个夜晚做出来的，重新打一遍光是想想就很折磨人。其实你不必这样。你可以导出自己的 Quizlet 卡组并迁移过来，再让应用把它们变成按 FSRS 排程的卡片。具体步骤见 [如何导出 Quizlet 卡组并把它们变成 FSRS 抽认卡](/blog/how-to-export-quizlet-sets-and-turn-them-into-fsrs-flashcards/)。如果你是从课堂笔记或一份 PDF 开始，AI 辅助制卡能从你的笔记和文件里生成一份初稿供你编辑，我在 [把你的笔记变成抽认卡](/blog/turn-notes-into-flashcards/) 里走了一遍这个流程。换工具并不意味着从零开始。

## 你拥有自己的卡组，而不是租用一个平台

付费墙让人不舒服，还有一个更深层的原因。当规则可以在你脚下随时改变时，你的学习系统其实从来就不真正属于你。你去年做的那套卡组一直好好的，直到免费上限再次变动的那一天。

换成开源应用，这种关系就反转了。你可以读代码，可以自托管。你的卡组不会被锁在一个随时能给你赖以使用的功能重新定价的平台里。随着时间推移，抽认卡会慢慢变成一张地图，记录你在学什么、又总是忘什么，这是非常私人的数据。我宁愿把它放在自己能掌控的地方。如果数据掌控是你四处寻找的主要原因，[用于间隔重复的自托管开源抽认卡应用](/blog/self-hosted-open-source-flashcards-app-for-spaced-repetition/) 讲了自托管这条路。

公平地说，Anki 同样免费，也同样认真对待间隔重复，很多人对它推崇备至。诚实的取舍是，它的界面更老派，配置起来也更需要耐心。Flashcards 瞄准的是同样免费、无订阅的思路，但用起来更现代，所以对那些想要 FSRS、又不想啃学习曲线的人，我一直把它推荐为 Quizlet Plus 替代方案。如果你想要一份并排对比，[Anki vs Quizlet vs 开源抽认卡应用](/blog/anki-vs-quizlet-vs-open-source-flashcards-app/) 把这三者放在一起比较，而早些时候那篇 [Quizlet 替代方案](/blog/quizlet-alternative/) 则更深入地讲了数据掌控、FSRS 和离线。

## 诚实的结论

如果你只是偶尔需要快速做一套卡片，免费版 Quizlet 仍然管用，连广告一起照单全收，我也不会假装不是这样。

但如果你寻找**免费的 Quizlet 替代方案**，是因为学习本身现在要花钱，那么你有一个真实的选项，它不会把你需要的部分锁起来。FSRS、离线学习、图片、AI 制卡，全部都在托管 beta 上免费，自托管则永久免费。没有刷到第四轮就被切断，试用结束时也没有订阅在那儿等着你。
