连续复习十张只考链式法则的抽认卡，可能会让链式法则显得很简单。可一旦碰上一道混合题，需要判断该用链式法则、乘积法则还是商法则，刚才的自信就消失了。这种落差，正是交错练习要揭示的。

要学会**如何用抽认卡进行交错练习**，先分别学习每个新类别，直到能够识别它，再在一轮检索练习中混合两三个彼此相关的类别。去掉会泄露类别的标签，翻面前先作答，卡片之后何时再次出现仍交给 FSRS 安排。不要只是为了让复习显得多样，就把毫不相关的牌组混在一起。

交错练习（interleaving）改变接下来出现的是哪类题目或卡片；间隔重复改变同一张卡什么时候再次出现。两者很适合配合使用，因为它们解决的是不同问题。

## 交错练习 vs 间隔重复：顺序和时间安排

[杜克大学给出的实用定义](https://arc.duke.edu/interleaving/)是：交错练习会在一次学习过程中有意混合不同主题；集中练习则把同一类内容放在一起。

如果 `A`、`B`、`C` 是三类相关题型，两种顺序可能是这样：

- 集中练习：`A A A B B B C C C`
- 交错练习：`A B C B A C A C B`

交错顺序会不断迫使你做选择。你必须先判断题型，再运用相应的解题步骤。在集中顺序里，上一张卡其实已经暗示了下一张该用哪类解法。

间隔重复处理的是另一个维度。随着记忆逐渐稳定，一张卡可能明天再次出现，四天后再出现一次，之后隔两周再回来。FSRS 会根据你的复习历史安排这些时间。

| 方法 | 它回答的问题 | 会改变什么 |
| --- | --- | --- |
| 主动回忆 | 我现在能不看背面，自己想出答案吗？ | 先作答，再看背面 |
| 交错练习 | 接下来需要调用哪类知识？ | 混合相关的卡片或题型 |
| 使用 FSRS 的间隔重复 | 这张卡应该什么时候再出现？ | 改变两次复习之间的间隔 |

到期复习队列里可以同时出现多个主题，但这并不等于在有意进行交错练习。它们可能只是碰巧在同一天到期。有效的交错练习需要一个理由，让这些类别值得相邻出现：它们容易混淆、需要在彼此竞争的规则中做选择，或者考试会要求你分辨它们。

很多受访学生并不知道这个术语。[2025 年一项针对两所大学 1,093 名本科生的调查](https://onlinelibrary.wiley.com/doi/full/10.1002/acp.70071)发现，学生对交错练习这一术语的了解程度很低，实际采用这种方法的也不多，不过其中一些人已经会混合新旧内容，只是不知道这个名称。

## 为什么集中练习感觉更好

集中练习会为每张卡省去一个判断步骤。如果前六道题都用了乘积法则，第七道大概率也是。练习表现很快就会变好，整个过程也很顺。但这种顺畅感反映的可能只是你对题目顺序的适应，而不是能否独立调用知识。

在[一项 2025 年的实验中，259 名参与者学习按六位艺术家给画作分类](https://www.sciencedirect.com/science/article/pii/S1041608025001803)。集中练习让他们在学习阶段表现更好，对学习效果的主观判断也更高；随机交错和自适应交错则在学习后的即时测试和延迟测试中取得了更好的成绩。在这项任务中，自适应顺序并不优于普通的随机交错。

这个结果并不能证明，打乱每副抽认卡牌组就能改善所有学科的学习。它清楚展示的是一个陷阱：练习时的表现和之后的学习成效可能并不一致，尤其是当任务要求区分相似类别时。

一项课堂研究更直接地把这一观点放到检索练习中考察。[2022 年一项针对 155 名高中理科生的研究](https://journals.sagepub.com/doi/10.1177/09567976211057507)在为期四周的实验中，每周把小测题按概念集中排列，或在不同概念之间交错排列。一个月后的测试中，通过交错小测练习过的概念，比通过集中小测练习过的概念记得更好。这些是课堂小测，并不是 FSRS 抽认卡队列，因此结果支持混合检索练习，却不能证明某种特定的应用内顺序一定有效。

当难度来自判断和主动回忆时，难度更高的练习可能有用。措辞含混、答案过长，以及毫无目的地随意切换语境，只是在制造阻力。

## 学习时如何混合主题：选择需要辨别的相近类别

最适合交错练习的内容彼此足够相关，因此你必须留意它们的差别。

适合放在一起的内容包括：

- 乘积法则、商法则和链式法则题
- 沿需求曲线移动与需求曲线发生位移
- 放在简短语境中的西班牙语简单过去时与过去未完成时
- 根据题目描述的结果，判断是有丝分裂还是减数分裂
- 输入相似、但行为保证不同的两种编程方法

不太合适的组合，是把一张西班牙语词汇卡、一种有机化学反应机理、一个报税截止日期和一条网络命令扔进同一堆。这确实增加了变化，却没有形成有用的比较。

[2019 年一项汇总 59 项交错练习研究的元分析](https://doi.org/10.1037/bul0000209)提醒我们必须准确看待这种方法。研究发现交错练习总体上有中等程度的优势，但结果高度取决于材料类型。视觉类别学习的收益更明显，数学任务的总体收益较小，说明文材料的结果并不明确，而以单词为材料的研究总体更支持集中练习。类别之间的相似度是一个重要的调节因素。

所以，真正实用的筛选问题不是“这些主题能不能打乱？”而是“在这些主题之间切换，是否会逼我判断该用哪条规则、哪个类别，或哪种解释？”

| 适合混合 | 目前先分开 |
| --- | --- |
| 你总会混淆的相似类别 | 你还没有理解的材料 |
| 会在同一场考试中出现的题型 | 长篇阅读材料或初次讲解 |
| 适用条件不同、彼此竞争的规则 | 没有共同判断点的不相关学科 |
| 同一区别的新旧例子 | 正面措辞已经泄露类别的卡片 |

## 集中练习 vs 交错练习：什么时候该切换

从第一个例子开始就完全采用交错练习，可能严苛得没有必要。如果你从没见过商法则，讲解才看一行就切换到别处，并不能教会你辨别题型，只会打断讲解。

从集中练习逐步过渡到交错练习，能保留两种形式各自有用的部分。

### 1. 先把一个类别学到能识别、能说清

读完讲解，再做几道同类型例题。目标是建立基本认识，不是追求速度。你应该能说清这条规则有什么作用，也能认出一个直观的例子。

对第二个类别重复这段简短的入门练习。只有当前两个类别不再陌生时，再加入第三个。

这一步很重要，因为已有知识水平可能改变练习效果。[2025 年一项预注册课堂研究纳入了 108 名德国三年级学生](https://www.sciencedirect.com/science/article/pii/S0959475225000702)，结果发现，用交错方式练习相似拼写规则带来的持久收益，包括把所学迁移到新单词上，主要出现在原有拼写基础中等到较高的儿童身上。孩子们在交错练习中也犯了更多错误，对学习成效的自我评价也更低。这项研究支持在该场景中使用交错练习，同时也提醒我们：它并不是任何学习任务都该采用的第一步。

### 2. 先建一小组易混类别

选两三个相关类别。先从一小批开始，比如每类三四张卡。这只是一个方便执行的参考量，并非研究得出的门槛。

这些可以是临时练习卡，也可以只是待筛选的题面。它们还不必进入你的长期复习队列。

如果有助于整理，可以把类别保留在元数据、笔记或准备清单里。但如果练习目标正是判断类别，就要把它从卡片正面去掉。

`链式法则：求导……` 已经泄露了该做的判断。

`求导……先用哪条法则？` 才会逼你自己选择。

### 3. 做一轮简短的混合练习

自己打乱这些临时卡，或手动排出一个会切换类别的顺序。如果 `A B C A B C` 的固定循环开始暗示答案，就别用这种可预测的排列。

看到每张卡的正面后都要先作答，再翻到背面。过程类卡片要先说出规则，再完成相应步骤；概念类卡片要先说出类别，再指出哪条线索把它和最接近的另一种答案区分开。

把这组卡当成辨别训练，不要当成排程内的额外复习。如果一张已经保存的卡还没到期，不要为了把它硬塞进这次混合练习而提交评分，否则会改变它的复习历史。练习时使用复制出来的题面，只有在正常的到期队列里才记录评分。

### 4. 把反复混淆的点做成对比卡

普通卡片检查一个答案。对比卡检查两个答案之间的边界。

例如：

- 正面：在其他条件不变时，某商品自身价格下降。这是需求曲线发生位移，还是沿需求曲线移动？
- 背面：沿需求曲线移动。商品自身价格变化会改变需求量；其他决定因素变化才会使需求曲线发生位移。

这张卡有价值，是因为它记录的是判断规则，而不只是 `需求曲线位移` 的定义。

背面要足够简短，让你能如实评分。[如何做出更好的抽认卡](/zh/blog/how-to-make-better-flashcards/)会继续讲如何设计卡片，[主动回忆与间隔重复](/zh/blog/active-recall-vs-spaced-repetition/)则解释为什么现在回忆答案和之后安排复习仍然是两项不同的工作。

### 5. 值得保留的卡片交给 FSRS 安排

练习结束后，如果某个区别以后仍然重要，或你反复答错，就把它保存成卡。交错练习不该成为忽略到期时间，或把每个主题都手动塞进每次复习的理由。一张内容清楚的卡片一旦值得进入长期复习，就如实作答，让 FSRS 决定它的下一个复习间隔。

下次想再检查辨别能力时，可以换一批新例子，再做一轮简短的临时混合练习。不必让每一道练习题都进入排程系统。

## 一组交错练习抽认卡实例

假设你正在学习三条微积分求导法则。先按类别集中练几道每种法则的直观例题，然后去掉类别标签，把卡片混在一起。

卡片 1：

- 正面：求 `(x² + 1)⁴` 的导数。先用哪条法则？
- 背面：链式法则。导数是 `8x(x² + 1)³`。

卡片 2：

- 正面：求 `x²eˣ` 的导数。先用哪条法则？
- 背面：乘积法则。导数是 `2xeˣ + x²eˣ`。

卡片 3：

- 正面：求 `(x + 1) / (x - 1)` 的导数。先用哪条法则？
- 背面：商法则。导数是 `-2 / (x - 1)²`。

卡片 4：

- 正面：求 `sin(x³)` 的导数。先用哪条法则？
- 背面：链式法则。导数是 `3x² cos(x³)`。

这组混合练习不只是增加了变化。每张正面都要求你先观察结构、选出法则，再开始计算。这更接近真正的考试，因为没人会在题目上方替你标注 `商法则`。

如果卡片 2 总是让你出错，可以再加一张对比卡：

- 正面：为什么 `x²eˣ` 要先用乘积法则，而不是链式法则？
- 背面：因为它是两个关于 `x` 的函数相乘，两个函数之间不存在嵌套关系。

不要保存四个几乎一模一样、只有表面变化的例子。一张清楚明确的对比卡，加上之后的混合练习，通常比一副塞满细微变体的牌组更有用。

## 让交错练习抽认卡真正要求主动回忆

正如上面的画作研究所示，交错练习可以帮助类别学习。但抽认卡本身仍要考查你真正想记住的内容。如果只看正面就能凭线索认出答案，而无须自己作答，那么即使把卡片混在一起，它也不会变成主动回忆卡。

带四个选项的卡片，可能只让你认出一句熟悉的话。标题里写着类别名称的卡片，可能已经告诉你该用哪种步骤。连续跟在五个近乎相同例子后面的卡片，也可能只是靠顺序记忆答出来。

用于交错练习的抽认卡，至少应该要求你自己回忆并说出下面一项：

- 答案本身
- 适用的类别或规则
- 把它和相近答案区分开的特征
- 这套步骤的第一步

这正是**交错练习与主动回忆**彼此补足的地方。主动回忆要求你实际尝试检索答案，交错练习则拿掉“下一次要做哪类尝试”的舒适提示。

当真实任务无法压缩到一张卡片时，模拟测试仍然重要。多步骤任务、时间控制和迁移能力要用混合题来练；抽认卡则用来记住这些题暴露出的细小规则和区别。[抽认卡与模拟测试](/zh/blog/flashcards-vs-practice-tests/)会更深入地解释这段衔接。

## 这些做法会让交错练习变差

把所有东西都混在一起，只会造成随机的上下文切换。不相关的类别并不会自动教会你某个共同区别，额外切换可能增加负担，却练不到任何有用的判断。

开始时机也很重要。学习者需要先有足够的理解，才能比较不同类别，而不是在陌生标签之间瞎猜。

类别提示会悄悄让整套练习失效。牌组名称、标题、重复措辞和固定顺序，都可能泄露接下来该给出哪一类答案。

不要只用练习速度评价这种方法。交错练习往往更慢，因为它多了一次判断，而这次判断本来就是练习的一部分。

不要轻信“自适应顺序或 AI 生成的顺序一定更好”这种说法。2025 年的画作研究没有发现其自适应顺序比随机交错更有学习优势。这个结果仅限于这一实验设计，但已经足以否定“个性化排序必然更优”的断言。

别让排程器卷入这场顺序实验。把交错练习和间隔重复结合起来时，应该保留 FSRS 的复习历史和到期时间。把有意设计的混合练习放在正常复习之外，不要只是为了维持整齐的交错模式，就不断重置或重新安排成熟卡片。

## Flashcards 在这套流程里负责什么

[Flashcards Open Source App](/zh/features/)为这套流程提供了清晰的长期记忆环节：明确的正面和背面、四档复习评分，以及用 FSRS 安排到期卡片。它公开的功能说明并没有承诺会在不同牌组或工作区之间自动执行交错练习，因此不要假设普通的到期复习队列会有意替你组成对比组合。

把这条边界保持清楚：

1. 为规则、区别和反复出错的点制作清楚的正反面卡。
2. 如果源材料很乱，可以在 AI 聊天中附上材料，让它起草候选卡，但每个答案都要自己核对。
3. 当你需要在多个类别之间做选择时，手动挑选一组相关题面来练习，最好换用新例子，并放在正常到期队列之外。
4. 只把真正暴露问题的错项，以及那些需要很久才能分清的区别做成卡。
5. 正常复习到期卡片，让 FSRS 更新它们之后的时间安排。

这样分工能避免一种常见的混乱：交错练习专心训练辨别，FSRS 队列专心负责长期记忆。

[FSRS 设置指南](/zh/blog/fsrs-settings/)会介绍目标记忆保持率和复习量，不用把折腾排程器变成一种业余爱好。准备好创建卡片、复习到期内容时，[入门指南](/zh/docs/getting-started/)会告诉你如何开始使用在线版应用。

## 最值得记住的一条规则

混合需要你做判断的内容，不要随机混合学科。规则真正陌生时，先做集中练习。能认出基本类别后，再进入交错练习。去掉标签，主动回忆答案，说清区别，把练习组控制在足够小的范围内，这样你才能看清自己为什么会错。时间安排则交给 FSRS。

这就是把**交错练习与间隔重复**结合起来时最清晰的分工：交错练习让你能在混合情境中选对知识，FSRS 则根据每张值得保留的卡片的复习历史和你的记忆保持率设置，安排它下一次出现的时间。
