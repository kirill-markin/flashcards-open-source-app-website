到了 2026 年，网上居然还有人在做 LSAT 抽认卡页面，劝大家继续复习 Logic Games。但现在的 LSAT 已经根本不考 Logic Games 了。自 2024 年 8 月的考试格式调整后，原来的 Analytical Reasoning section 已经被第二个计分的 Logical Reasoning section 取代。当前 LSAT 的核心是 [Logical Reasoning and Reading Comprehension](https://www.lsac.org/lsat/prepare/types-lsat-questions)，而 [Argumentative Writing](https://www.lsac.org/lsat/prepare/types-lsat-questions) 则单独处理。LawHub 在介绍 [redesigned LSAT PrepTests](https://app.lawhub.org/article/redesigned-official-lsat-preptests-available-now) 的说明里，也直接写明了这次变化。

这点很重要，因为旧的备考建议会一直挂在网上，新的考试周期却不会等你。LSAC 明确写着，[2026-2027 LSAT testing year 的报名会在 5 月中旬开放](https://www.lsac.org/lsat/register-lsat)。如果你的牌组还围着 Logic Games 模板、过时的 section 设定，或者一堆泛泛的词汇卡打转，那你练的就不是现在这场考试。抽认卡不需要看起来很厉害。它们需要训练的，是这一版 LSAT 真正奖励的判断。

## 按当前 LSAT 建卡，不要按互联网对旧版考试的记忆建卡

官方考试结构本身，就应该决定你往牌组里放什么。

LSAC 当前的题型页面写得很清楚：LSAT 的选择题部分包含 Reading Comprehension 和 Logical Reasoning，另有一个不计分的 variable section，类型也只会是这两者之一；此外还有一个单独、不计分的写作样本，叫做 [LSAT Argumentative Writing](https://www.lsac.org/lsat/prepare/types-lsat-questions)。LawHub 关于 2024 年 8 月格式调整的说明则更直接：不再有 Analytical Reasoning，不再有计分的 Logic Games，现在是两个计分的 Logical Reasoning sections。

这意味着，2026 年真正有用的 **LSAT flashcards**，通常更应该偏向：

- Logical Reasoning 的论证模式
- 你在 Reading Comprehension 里反复踩中的陷阱
- conditional logic 的翻译错误和推理错误
- 来自官方练习的错误选项模式
- 复盘时不断重复出现的时间分配和执行失误

它们应该尽量远离：

- 庞大的词汇型牌组
- 为当前考试准备的旧版 Logic Games 模板
- 做完就再也不会回看的篇章摘要
- 仅仅因为书页看起来重要，就从备考书里整段抄下来的卡片

如果你是在这个考试周期稍晚一些的时候参加考试，还有一点要分清：LSAC 已经宣布了 [2026 年 8 月 LSAT 的交付方式变化](https://www.lsac.org/blog/evolving-how-we-deliver-lsat-increase-test-security-and-test-taker-success)，包括逐步转向 in-center testing 和新的 LawHub 交付平台，但这说的是考试管理和界面变化，不是题型结构又变了。

## 好的 LSAT 抽认卡保存的是判断，不是章节标题

LSAT 复习很容易制造一种很特定的假自信。你读完一道题的解析，点点头，觉得自己懂了。可下一组计时练习只是稍微换个说法，同样的错误又回来了。

这也是为什么 **flashcards for LSAT** 在每张卡都只保存一个可复用动作时效果最好。这个动作应该来自真实错题，或者某个会反复出现的判断：

- 哪个线索说明这个选项超出了题干范围
- 哪一句才是真正的结论
- 什么措辞触发了 contrapositive
- 为什么某个 inference 走过头了
- 哪类 Reading Comprehension 选项听起来很漂亮，却依然没有对准文章

弱卡片通常只会写成主题标签：

- `strengthen questions`
- `main point`
- `conditional logic`
- `parallel flaw`

这些是笔记，不是高质量的提取提示。

如果你眼下最大的瓶颈是卡片质量本身，那在继续加卡之前，更值得先看这篇配套文章：[2026 年如何做出更好的抽认卡](https://flashcards-open-source-app.com/blog/how-to-make-better-flashcards/)。

## LSAT Logical Reasoning 抽认卡应该跟踪论证动作和陷阱选项

这部分是抽认卡最能发挥作用的地方，尤其是现在的 LSAT 有两个计分的 Logical Reasoning sections。

很多糟糕的 **LSAT Logical Reasoning 抽认卡** 看起来就像缩成几行的教材目录：

- 正面：`Necessary assumption`
- 背面：一整段解释 necessary assumption 是什么

这和你在 Logical Reasoning 里真正面对的压力并不匹配。在时间压力下，真正值得记住的目标通常更小：

- 这道论证真正依赖的是哪一道缺口
- 什么样的答案会 strengthen 或 weaken 这个缺口
- 哪个错误选项看起来相关，但其实根本没有动到结论
- 哪些措辞提示这是 principle、flaw 或 method question
- 你漏掉的是哪种 conditional assumption 或 causal assumption

更强的提示会像这样：

- 哪个线索说明作者把 correlation 当成了 causation？
- 为什么这个很诱人的答案仍然没有影响结论？
- 哪个措辞让这句话从“某些情况”变成了“所有情况”？
- 为什么这个答案只是重述了证据，而不是真正的 weaken answer？

一组好的 **LSAT Logical Reasoning 抽认卡**，最后看起来通常不像概念词汇表，反而更像一套持续修正判断的资料库。

## LSAT Reading Comprehension 抽认卡应该来自重复错法，而不是每一篇文章

Reading Comprehension 很容易让人走向两个极端：做得太少，或者做得太多。

做得太少的典型想法是：“RC 不就是阅读吗？抽认卡应该帮不上什么。”

做得太多则会变成：每段一张卡、每篇一张卡、每一组 comparative reading 再加一张超长摘要。

我会停在中间。好的 **LSAT Reading Comprehension 抽认卡**，通常来自这些会反复出现的错法：

- main-point 陷阱
- inference 题里总会多推一步
- 你总把 author attitude 的措辞压得太平
- function question 里你记住了事实，却漏了这一段为什么存在
- comparative reading 里你把两个观点糊成一个模糊的大意

真正的规则其实很简单：给“模式”做卡，不要给“篇章”做卡。

你不需要一副永久保存、里面全是某个陌生话题一次性摘要的牌组。你需要的是一套 **LSAT Reading Comprehension 抽认卡**，专门处理那些一直在让你丢分的阅读习惯。

## LSAT conditional logic 抽认卡应该保持抽象，而且可复用

conditional logic 是 LSAT 上最适合用抽认卡处理的部分之一，因为同一种错误总会换个外衣再回来。

但这也是很多人最容易把牌组写得过于字面的地方。他们会把几乎整段专有题干贴进卡片里，然后记住的是那一道题，而不是底层逻辑。

我会让 **LSAT conditional logic 抽认卡** 只盯着那些可复用的动作：

- 如何翻译 only if、unless、except、until
- 如何分清 sufficient condition 和 necessary condition
- 如何干净地写出 contrapositive
- 如何识别 invalid reversal 和 invalid negation
- 如何串联条件句，而不漏掉中间一环
- 如何注意到 quantifier 悄悄改变了逻辑范围

例如：

- 在做其他事之前，最稳妥的 `unless` 翻译方式是什么？
- 这里到底是哪一步导致了 invalid reversal？
- 两条 conditional rule 可以连接时，哪一个中间项能让你把它们串起来？
- 哪个线索说明这里需要用 contrapositive，而不是直接改写原句？

这样你得到的牌组才是真的能反复使用。它也能让你的笔记继续停留在题目底层逻辑上，而不是把整个学习系统变成一份抄来的官方材料档案。

## 建一份你真的会再打开的 LSAT 错题本

大多数人都会做某种形式的错题记录。真正会把 **LSAT 错题本** 做到能持续复习下去的人，要少得多。

最常见的失败方式很熟悉：

- 截图题目
- 往笔记里倒一大段说明
- 告诉自己以后再复习
- 以后从来没有再复习

抽认卡有用，是因为它强迫你压缩信息。把一条 **LSAT 错题本** 记录缩到下面这些内容时，价值通常会立刻提高：

- 这次到底是哪种错误
- 你漏掉了哪个线索
- 为什么那个诱人的选项是错的
- 下次要改什么，才会选出不同答案

我会按功能打标签，而不只是按 section：

- `lr-gap`
- `lr-trap-answer`
- `rc-main-point`
- `rc-overreach`
- `conditional-logic`
- `timing-miss`

这样一来，你的错题本就有了第二次生命，不再只是“存放好意图”的坟场，而是一个真正的复习系统。

如果你现在最好的卡片本来就主要来自练习复盘，这篇文章会把那一步讲得更细：[2026 年如何把练习题变成抽认卡](https://flashcards-open-source-app.com/blog/how-to-turn-practice-questions-into-flashcards/)。

## 一套实用的 LSAT 每周抽认卡工作流

我会故意把这套流程做得很朴素。

1. 做一组计时的 Logical Reasoning、Reading Comprehension，或者一组混合复盘。
2. 只标出那些暴露了可复用错误的错题。
3. 在你去读三篇不同解析之前，先自己写一句短说明：到底哪里出了问题。
4. 把这句说明改写成一到两张候选卡。
5. 按 section 和模式打标签，而不只是按来源分类。
6. 含糊的卡片尽快删除。
7. 每天复习到期卡，并让新卡数量始终低于你的雄心。

这会比试图把接触过的所有内容都塞进一副英雄式大牌组里，更像一套健康的 **LSAT 间隔重复** 工作流。

## 使用合法的个人笔记和总结，不要复制受保护的专有题目

这一点应该明确写出来。

Flashcards Open Source App 不是 LSAC 的官方工具，也没有任何官方 LSAT 集成。请只用它处理你自己的笔记、你自己的总结、你自己的解释，以及你对合法来源材料的复盘。不要用它去建立一份你无权复制的 LSAT 题目、篇章或答案库私有档案。

这其实也是更好的学习方式。最强的卡片通常不是整道题。更强的，往往是你自己压缩出来的那句总结：这次推理错在哪里，下次希望自己做出什么不同判断。

## 先把牌组变小、变利落，FSRS 才真的帮得上忙

很多人会把这一步的顺序做反。

他们先造出一副臃肿的大牌组，再寄希望于调度器以后来拯救它。

我喜欢 **FSRS LSAT 备考** 这类工作流，是因为你的错误分布本来就不均匀。某一种 flaw pattern 两次复习就稳了。某一种 conditional translation mistake 会连着回来四次。某种 Reading Comprehension overreach 每次看都像是显而易见，直到下一篇难文章又把你带偏。

这正是间隔重复最擅长处理的那种提取模式。

但 FSRS 做不到的，是拯救含糊的卡片，或者已经过时的牌组。所以我会把顺序维持得很简单：

1. 按当前 LSAT 的结构建卡
2. 把错题压缩成非常窄的提示
3. 把 LR、RC、conditional logic 和错误选项模式分开到足够清楚，方便复习
4. 尽快删掉弱卡
5. 再让 FSRS 去安排留下来的卡片

## 建一副能帮你更早看见错误的牌组

当前的 LSAT 不再要求你记住旧版 Logic Games 的世界。它要你做的是：细读文本、跟住论证、处理 conditional logic 时不松手，以及别再一次次被那些精致的错误选项骗走。

这通常意味着：

- 卡片会比你以为的更少
- 错误选项模式会比你以为的更多
- conditional logic 的清理工作会比你想做的更多
- 篇章摘要会比网上建议的少得多

这才是我会在 2026 年信任的 **flashcards for LSAT**：针对当前考试格式，比你的笔记更窄，而且完全建立在你真正会重复犯的错误之上。
