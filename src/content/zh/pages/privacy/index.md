## 我们收集的信息

当您使用托管云端测试版时，我们会保存您用于身份验证的电子邮箱地址，以及提供服务所需的学习数据，包括卡片、复习记录和工作区元数据。对于自托管实例，除非您自行部署的版本会主动将数据发送给我们，否则相关数据不会传输给我们。

## AI 功能

如果您选择在托管应用中使用 AI 聊天功能，您输入的提示内容、完成该请求所需的卡片相关上下文、上传的文件和图片，以及语音输入或转录请求，可能会被发送至服务器当前配置的第三方 AI 提供商。具体使用哪家提供商，取决于发起请求时托管服务器当时的配置。

## 连接外部 AI 客户端

当您通过远程 MCP 服务器（`mcp.flashcards-open-source-app.com/mcp`）或 Agent API 将 Flashcards 连接到外部 AI 客户端时，您所请求的卡片数据会被发送给该客户端以及运营该客户端的 AI 或模型提供商。此类处理受该提供商的条款约束，而非我们的条款。

## 我们如何使用您的数据

我们会将您的数据用于提供身份验证、云端同步以及可选的 AI 功能。我们不会出售您的数据，也不会将其用于广告。身份验证和工作区访问由公开仓库中说明的后端服务处理。

## 数据存储

托管云端数据存储在 AWS 基础设施中，其中主要应用数据保存在 Postgres。数据在传输过程中会加密。我们会在您的账户处于活跃状态期间保留您的托管学习数据，并在您删除账户或请求删除时将其删除。运维日志仅在运行和调试托管测试版所必需的期限内保留，通常不超过 90 天。

## Cookie

我们会在登录流程中使用 `session`、`refresh` 和 `logged_in` 等身份验证 Cookie。网站正常运行不依赖跟踪 Cookie 或第三方分析工具。

## 数据删除

对于自托管实例，您可以直接控制数据库。对于托管应用，您可以在 iOS 应用或网页应用中删除账户；您也可以通过 Agent API 或已连接的 MCP 客户端请求删除托管数据；如果在删除托管数据时还需要额外协助，也可以联系支持团队。
